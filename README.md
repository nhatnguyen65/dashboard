# [Dashboard]

## Hướng Dẫn

Sau khi clone dự án về mở Terminal tại thư mục gốc của dự án:
```
  - Chạy lệnh để cài các thư viện:
    npm install
  - Chạy Website ở Local bằng LiveServer hoặc lệnh bên dưới:
    npm start
  - Mở Terminal thứ 2 để chạy dữ liệu tạm xem cách Webstie lấy dữ liệu:
    npm run server
```
